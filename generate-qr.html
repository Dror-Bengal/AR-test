QR Code Generator

Enter the URL where your AR experience is hosted, then generate a QR code to print or display on a billboard.

How to deploy:
1. Host ar-experience.html on any web server (GitHub Pages, Netlify, Vercel, etc.)
2. Paste the URL above and generate the QR code
3. Print the QR code and place it on/near the billboard
4. When someone scans it, their camera opens and the AR figure jumps out!

Quick local test:
Run npx serve . in the project folder, then use ngrok or similar to get a public URL.
